Reingorcement Learning

分类1

model-feel rl 不理解环境

  • Q Learning
  • Sarsa
  • Policy Cradients

model-based rl 理解环境

比model-free多了一个虚拟环境，model-free中的方法也适用

分类2

Policy-based RL 基于概率

  • policy gradients et ta.

Value-Based RL 基于价值

对连续的动作无能为力

  • Q learning
  • Sarsa

Actor-Critic

Actor基于概率生成动作，Critic根据给定的动作计算价值

分类3

Monte-Carlo update 回合更新

游戏开始->游戏结束->更新

  • 基础版 policy gradients
  • Monte-Carlo Learning

Temporal-Difference update

游戏开始->更新->更新->…->游戏结束
每一步都会更新，边玩边更新，更有效率，大部分增强学习算法都是单步更新方法

  • Q Learning
  • Sarsa
  • 升级版Policy Gradients

分类4

On-Policy 在线学习

自己玩，边玩边学习

  • Sarsa
  • Sarsa($\lambda$)

Off-Policy 离线学习

既可以自己玩，也可以看着别人玩

  • Q Learning
  • Deep Q Network
  • RL
  • Reingorcement Learning
